NubiaNova Trading Academy
Milestone #1 & #2 project files.

How to use:
1. Extract this ZIP.
2. Upload the individual files (not the ZIP) to your GitHub repository.
3. Ensure index.html is at the repository root for GitHub Pages to serve it.
4. Edit files directly on GitHub or locally and push updates.

Files included:
- index.html
- indicators.html
- risk-management.html
- concepts.html
- references.html
- about.html
- quiz.html
- quiz.js
- style.css
- images/ (create your own images folder and add files if needed)

Notes:
- Replace placeholder contact/email and images before submission.
- Validate HTML using the W3C validator if desired.
